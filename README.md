Full Stack Game Library
A full-stack web application that allows users to browse and manage a game library.  
Built using PHP, MySQL, HTML, CSS, and JavaScript.
Features
User System
- User registration and login
- Secure password hashing
- Session-based authentication

Game Library
- Display games with images and genres
- View individual game details
- Responsive card-based UI

 Admin Dashboard
Add new games
 Edit existing games
Delete games
Upload game images
  lean sidebar-based dashboard interface

Search (AJAX)
- Live search without page reload
- Autocomplete suggestions while typing
- Dynamic results update
 Technologies Use
- **Frontend:** HTML, CSS, Bootstrap, JavaScript (jQuery AJAX)
- **Backend:** PHP
- **Database:** MySQL
- **Version Control:** Git & GitHub
